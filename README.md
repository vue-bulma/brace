# Brace

Code editor powered by Vuejs and browserify Ace (Brace)

## Installation

```
$ npm install vue-bulma-brace --save
```


## Examples

```vue
<template>
  <div>
    <brace style="height: 500px" 
      :fontsize="'12px'" 
      :theme="'github'" 
      :mode="'json'"
      :codefolding="'markbegin'"
      :softwrap="'free'"
      :selectionstyle="'text'"
      :highlightline="true"
      :editorValue="'Hello World!'">
    </brace>
  </div>
</template>

<script>
import Brace from 'vue-bulma-brace'

export default {
  components: {
    Brace
  }
}
</script>
```
